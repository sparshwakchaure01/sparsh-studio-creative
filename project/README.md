# DesignStudio - Freelance Graphic Designer Portfolio

A modern, clean, and professional static website for freelance graphic designers. Built with React, TypeScript, and Tailwind CSS.

## Features

- **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- **Static & Fast** - No database, no backend, lightning-fast loading
- **Modern Design** - Clean, professional, corporate-friendly aesthetic
- **Smooth Navigation** - Smooth scrolling between sections
- **Contact Integration** - WhatsApp and email contact options
- **Portfolio Showcase** - Grid layout for showcasing designs
- **Digital Shop** - Section for selling templates and digital products
- **SEO Optimized** - Proper meta tags and structure
- **Easy to Customize** - Beginner-friendly code structure

## Pages/Sections

1. **Home** - Hero section with call-to-action buttons
2. **Services** - Detailed service listings with pricing
3. **Portfolio** - Gallery of design work
4. **Shop** - Digital products and templates
5. **Contact** - Contact form with WhatsApp integration
6. **Footer** - Links and social media

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Customization

See `CUSTOMIZATION_GUIDE.md` for detailed instructions on how to customize:
- Contact information
- Social media links
- Brand name
- Pricing
- Services
- Portfolio items
- Digital products
- Colors and styling

## Important Customization Steps

Before deploying, make sure to update:

1. **Contact Information** - Replace `YOUR_PHONE_NUMBER` and `your.email@example.com`
2. **Social Media Links** - Update links in Footer component
3. **Brand Name** - Replace "DesignStudio" with your business name
4. **Page Title** - Update title and meta tags in `index.html`
5. **Pricing** - Adjust service and product pricing
6. **Portfolio** - Replace placeholder designs with your actual work
